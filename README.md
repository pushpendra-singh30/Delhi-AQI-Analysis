# Delhi AQI Analysis

## Project Overview

This project presents an in-depth analysis of Delhi Air Quality Index (AQI) data using Python, Pandas, Matplotlib, and Seaborn.

## Research Questions

1. Which pollutants contribute most to poor air quality in Delhi?
2. What relationship exists between PM2.5 and other pollutants?
3. How do pollutant concentrations vary over time?
4. Which pollutants are most strongly correlated?
5. What environmental challenges can be identified from AQI data?

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Key Findings

* PM2.5 and PM10 are the major pollutants affecting Delhi's air quality.
* PM2.5 has a strong positive correlation with PM10, CO, and NO.
* Pollution levels show significant fluctuations over time.
* Vehicular emissions and dust pollution are major environmental challenges.
